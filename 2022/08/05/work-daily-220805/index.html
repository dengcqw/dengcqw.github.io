js 比较boolean的疑惑

下面写法对吗？状态一直不对
const [editable, setEditable] = React.useState(props.editable?true:false)
if (editable){} 和 editable?: 的写法只判断null/undefined对象。不判断boolean，boolean对象也是对象。需要直接比较true/false。
